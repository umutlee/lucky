## 文檔維護原則

1. **更新及時性**
   - 代碼變更時同步更新相關文檔
   - 定期檢查文檔的準確性

2. **內容規範**
   - 使用繁體中文撰寫
   - 保持文檔結構清晰
   - 提供具體的代碼示例

3. **版本控制**
   - 重要更新記錄在 CHANGELOG.md
   - 標註文檔的最後更新時間

4. **文檔評審**
   - 重要文檔變更需要評審
   - 定期進行文檔審查
